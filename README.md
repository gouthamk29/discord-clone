# Discord clone

Online live Chat application clone of discord.

## Features
*Real time chat using socketIo
*Polling of chat as fallback
*use live-kit for video calls


---

## Tech Used
- React
- NextJs

##Start guide

Genearte prisma 
```bash
npx prisma generate 
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
